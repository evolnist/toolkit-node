特長・こだわり

チャイルドガーデンは、女性(お母さん)目線で立ち上げた、女性が運営する保育所です。
私たちは、お子さまをお預かりするだけではなく、お子さま一人一人の個性とペースを何よりも大切にし、発見の力を一緒に育んで行きたいと考えております。
また、保護者の方に安心してお子さまをお任せいただけるよう、週1回ミーティングを行い、お預かりしているお子さまの成長・状態をすべてのスタッフが把握できるよう努めております。
経験豊富なスタッフを揃えておりますが、保育所としてはまだまだ歩き始めたばかりです。みなさまにはお力をお借りすることもあるかと思いますが、常に前向きに、良い保育をご提供できるようがんばりますのでよろしくお願いします。

徹底した衛生管理

徹底した衛生管理で、大事なお子さまをお守りします。

チャイルドガーデンでは大切なお子さまを守るために、徹底した衛生管理を行っております。保育所は常に清潔な環境を整え、掃除もすみずみまで丁寧に行います。

口にいれてしまうようなおもちゃは毎日水ふきし、週末には全てのおもちゃを消毒、お布団なども日干しを行い、ばい菌やウイルスにも最大の注意を払っています。

暑い時期には、散歩帰りにお子さま一人一人の体をタオルで拭いています。あたりまえの衛生管理を徹底することで、すこやかなお子さまの健康と安全をお守りします。

【職員の衛生管理】

  • 職員全員の保菌検査（赤痢、腸チフス、パラチフス、サルモネラ、O－157）を原則、毎月行なっています。
  • 職員健康診断を毎年1回実施しています。

安全を重視したおもちゃ

安全を重視したおもちゃで、お子さまが楽しく過ごせます。

おもちゃにもチャイルドガーデンのこだわりが！舐めてしまっても安全な“木”のおもちゃなどを中心に、触れて感性を鍛えるおりがみ、ぬりえ、ブロックなど、社会性や想像力を高め、お子さまが毎日元気で楽しく過ごせるようなおもちゃを充実させています。

また保育士が教えてくれる、“てあそび”など手先を使った遊びで脳の発達も助け、風船作りや絵本の読み聞かせ会など多彩なお遊びを行い、お子さまが毎日自然に通いたくなるそんな保育所になるよう努めています。

コミュニケーションを大切に

お子さまの笑顔のために、コミュニケーションを大切にしています。

着替えなど、自分のことは自分で少しずつゆっくりできるように、お子さまの成長に合わせたコミュニケーションを大事にしていきます。また、お昼寝とお遊びの空間を完全に分けることによりお子さまが落ち着いてゆっくりと過ごせ、すこやかな成長を助けます。

子育てに悩むママたちが、保育士に気軽にご相談できる環境が整っています。お子さまの成長をママと一緒になって確かめていく、そんなお子さまにもママにもあたたかい保育所でありたい、それがチャイルドガーデンの願いです。

楽しく過ごせるイベント

一年中、子どもたちが楽しく過ごせるイベントも充実。

毎週「リズム教室」「英語教室」なども取り入れて、お子さまが耳や体でリズムや音感、英語を身につけるカリキュラムも行っています。
習い事感覚で一時保育の利用も可能です（週1、週2のコースもございます）。

また季節を感じられるような、七夕やクリスマス会など簡単なイベントも実施。ボールすくいや紙で作った玉入れ、節分には豆まきなど、手先を使ったお遊びをみんなで楽しみます。

お誕生日には保育士からお誕生日カードや折り紙で作った手作りメダルのプレゼントも！ママにもお子さまにもアットホームな雰囲気がチャイルドガーデンの魅力です。

おいしいお弁当

年齢や体質に合わせて選べるので安心です。

チャイルドガーデンのお弁当は、お子さまの年齢やアレルギーなどの体質に合わせて選べます。

離乳食は前/中期、後/完了期の2種類、通常のメニューにもアレルギー対応（卵、乳）があります。うれしいデザート付きです。
※離乳食メニューは卵、乳を使用しておりません。
※アレルギー対応メニューは、通常メニューと主食・副食が異なります。

先生も同じメニューを食べています！

 

【安心・安全なお弁当】

毎月、献立表をご用意しており、代表的な野菜については食材の産地も記載しております。
一部の食品（魚、エビなど）を除いて国産品を使用しており、食品・食材ともに中国製（産地）のものは一切使用しておりません。（ただし、調味料に含まれる微量の原料は除きます。）

【衛生的な環境】

お弁当は行政指定工場で作っており、保健所から不定期（抜き打ち）で検査を受けています。
製造工場の検査項目は施設設備、食品の保管など27項目、お弁当の検査項目は大腸菌郡、ボツリヌス菌、黄色ブドウ球菌、O-157、残留農薬、保存料、遺伝子組換えの有無、酸化、過酸化の9項目です。

【食べ残しを減らす工夫】

健康面だけを重視すると、野菜類の食べ残しが多くなることも多いため、形、大きさ、鮮度（生鮮、冷凍）を使い分けています。
例えば、人参などは大きさ・味付け・鮮度により９種類から使い分けます。食べ残しの多いピーマンは、献立に合わせて苦味の少ない冷凍品を使用することもあります。人気の果物も皮をむいておくなど、食べやすいように工夫しています。
